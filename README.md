# Concert Crowd Simulator

Interactive concert crowd simulator with YouTube integration. Control the crowd energy, vibe, and size while playing your favorite tracks.

## Features

- **YouTube Integration**: Play any YouTube video or use preset tracks
- **Interactive Controls**: Adjust crowd size (5-50 people) and energy level (0-100%)
- **Crowd Vibes**: Switch between Chill, Hype, and Mosh pit moods
- **Festive String Lights**: Twinkling lights across the stage
- **Responsive Design**: Optimized for both desktop and mobile
- **Accessibility**: Supports reduced motion preferences

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/hereshecodes/concert-crowd-sim.git

# Navigate to the project directory
cd concert-crowd-sim

# Install dependencies
npm install

# Start the development server
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000).

## Usage

1. **Select a Track**: Click a preset button or paste a YouTube URL
2. **Adjust Crowd Size**: Use +/- buttons to add or remove crowd members
3. **Set Energy Level**: Control how energetic the crowd is
4. **Choose a Vibe**: Pick from Chill, Hype, or Mosh modes
5. **Play/Pause**: Control the crowd animation

## Deployment

Deploy to GitHub Pages:

```bash
npm run deploy
```
